\section{Introduction}
\label{sec:intro}

The use of discontinuous Galerkin method for nonlinear hyperbolic problems was propelled to fore-front by the work of \citear{cockburnShu1998a}. These class of methods combine the desirable qualities of numerical fluxes studied in quite detail for the FVM along with the advantages of FEM in obtaining high order solutions on complex geometries \cite{cockburnShu2001}.

In this work, the performance of DG methods for Euler equations is studied. It has been demonstrated amply that high order polynomials perform better on a given grid resolution, irrespective of the flux scheme employed (see for example, \cite{qiuKhooShu2006}). This work tries to see whether the high order polynomials perform better even on coarser grids. The number of degrees of freedom is kept constant across the polynomial degrees since it approximately represents the memory load on CPU. Similar attempt has been made by [Yano \& Darmofal, 2012] where error in turbulent drag coefficient is compared on adaptive mesh. \citear{nogueiraFelguerosoColominas2009} compare solution accuracy of DG and FVM using the same grid resolution. Here the FVM grid was generated by connecting the nodes of DG method which yields the same approximate grid size, although the dof are lower in FVM case. Similarly, in [Ursachi et al 2021], CG and DG method using adaptive grid for turbulent flow simulations. [Yu et al 2014] compare DG, spectral diff and correction reconstruction methods on smooth test cases in linear advection and Euler equations.

All the previous works focus on flows which are dominated by smooth variations, if not entirely smooth. A comparison of solutions within the DG method for different polynomial degrees on discontinuity-dominated test cases is so far not done according to the authors' knowledge. In this work, the effect of polynomial order on certain 1d and 2d test cases is analysed within a given numerical methodology.
